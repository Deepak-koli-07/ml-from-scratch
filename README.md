# ML From Scratch – Intuition & Practice

This repository contains structured notebooks covering core Machine Learning and Deep Learning concepts, with a focus on understanding algorithms from first principles.

The emphasis is on:
- Clear intuition behind each method  
- Step-by-step derivations and logic  
- Simple, interpretable math  
- Visual and conceptual clarity  

These notebooks are designed to build a strong conceptual foundation rather than only demonstrate library usage.

## Purpose

This repository focuses on:

- How algorithms work internally  
- Why each step in training exists  
- How models learn and adjust  
- How decision boundaries and errors evolve  

It serves as a personal knowledge base and reference for foundational ML concepts, useful for both practice and interview preparation.

Each notebook is written with clarity and rigor, aiming to make complex ideas precise and accessible.
